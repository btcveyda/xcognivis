# React Native Mobile App
Android-first cross-platform mobile application.

## Prerequisites
- Node.js 16+
- Java Development Kit (JDK 11+)
- Android SDK

## Setup
```bash
npm install
```

## Running on Android
```bash
npm run android
```

## Development
```bash
npm start
```

## Project Structure
- `src/screens/` - Screen components
- `App.tsx` - Main navigation
